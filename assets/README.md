# Assets Directory

This folder contains static assets for the Crest Consultancy website.

## Structure

```
assets/
├── images/          # Image files (logos, photos, icons)
│   ├── logo/        # Brand logos
│   ├── team/        # Team member photos
│   └── clients/     # Client logos
└── README.md        # This file
```

## Image Guidelines

- **Format:** Use WebP for photos, SVG for logos/icons
- **Optimization:** Compress images before adding
- **Naming:** Use lowercase with hyphens (e.g., `hero-background.webp`)
